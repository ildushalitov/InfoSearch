# Задание 2 

## DeploymentManual
1. Установить spacy:
```bash
pip install spacy
```
2. Запустить text_processing.py

- скрипт проходится по файлам в **../task1/cleaned** и обрабатывает их 
- результаты сохраняются в папку **processed_txts.py** в формате **<doc_id>_tokens.txt** и **<doc_id>_lemmas.txt**
